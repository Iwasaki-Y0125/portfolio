# Portfolio

## 概要
このリポジトリは、私 Iwasaki Yuka が制作したポートフォリオサイトです。
HTML、CSS、JavaScript を用いて基本的な構造・スタイル・動きを実装しています。

公開サイト: [https://iwasaki-y0125.github.io/Portfolio/](https://iwasaki-y0125.github.io/Portfolio/)

## 目的
- 自分自身のプロフィールや学習内容、作品を整理・公開する場として制作
- Web開発（フロントエンド）の学習をアウトプットとして経験する
- 今後のスキルアップ（UI/UXデザイン、Webアプリ開発など）に向けた第一歩として記録

## 技術スタック・動作環境
- HTML5
- CSS3（レスポンシブ対応、レイアウト、カラー調整）
- JavaScript（インタラクションや動的要素）
- GitHub Pages による静的サイトホスティング

## ファイル構成
https://github.com/Iwasaki-Y0125/Portfolio/blob/main/structure.txt

- `index.html`：トップページ。プロフィールや作品紹介を掲載
- `css/`：スタイルシート。デザインとレイアウトを管理
- `js/`：JavaScript。ユーザー操作に応じた動的挙動を実装
- `image/`：画像ファイルを格納
- `partials/`：HTML の共通部分（ヘッダー、フッター、ナビゲーションなど）を管理

## 学びのポイント・工夫
- HTML、CSS、JavaScript の基礎を理解することを意識
- レスポンシブ対応でスマホ・PCでの表示を調整
- ナビゲーションや作品紹介セクションを見やすく、操作しやすく設計
- 自分の趣味や関心（ドット絵）をデザインに反映

## 今後の展望
- GitHubで継続的に更新し、成長の記録を残す
